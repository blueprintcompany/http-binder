# Security Policy

## Supported versions

Security fixes target the latest public version of `BlueprintSoftware.HttpBinder`.

## Reporting a vulnerability

Please report security issues privately by emailing the maintainer or using GitHub private vulnerability reporting if it is enabled for the repository.

Include:

- affected package version
- minimal reproduction or proof of concept
- expected impact
- any known mitigation

Do not open public issues for exploitable vulnerabilities until a fix or mitigation is available.
